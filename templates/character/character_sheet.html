Ward DC by Attribute

Ward = 5 + Attribute + your highest-ranked skill governed by that attribute (magic_overview.md). This is what Spell Overcomes (and other effects) roll against.

Skills

BASE = ranks bought from your skill point budget. MOD = free modifiers (career/racial/feat). TOTAL = BASE + MOD + governing Attribute, the bonus added to 1d12 for that skill check. Trained: the Attribute only applies once BASE is 1 or higher - a skill with 0 Base is untrained and rolls without it.

Armor

Check "Worn" for the armor piece(s) currently equipped - only worn armor contributes to AR / Armor Penalty above. AR is your armor's Durability: it starts equal to Max AR and drops by 1 each time you're hit (armor.md); Max AR is the original value used as the repair cap.

Magic Skills by School

Skill Total = Base + Mod + governing Attribute, same as the Skills tab (Trained: the Attribute only applies once Base is 1 or higher). Your Spell Modifier (magic_overview.md), used for Spell Attacks vs. Evasion and Spell Overcomes vs. Ward, is this Skill Total minus Armor Penalty.

SchoolAttributeSkill TotalArmor Penalty

Spells Known

Each card holds one spell's header fields plus a full-width Effect box for the actual rules text. Preparation tracking is derived from the spell's School (magic_overview.md): Arcane spells are only castable from a scribed scroll, so track how many you're carrying; Divine spells are always known and castable by spending Mana, so just toggle whether you've already used it since your last hour of prayer.
